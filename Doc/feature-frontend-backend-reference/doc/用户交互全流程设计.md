## 1. 整体架构概览

### 1.1 三大核心组件

```
┌─────────────────────────────────────────────────────────────────────┐
│                        MemoryLib2.0 主界面                           │
├─────────────────────┬───────────────────────┬─────────────────────┤
│                     │                       │                     │
│   Panel (数据面板)   │   Canvas (任务画布)    │  Chatbot (智能助手)  │
│                     │                       │                     │
│  - 数据可视化        │  - 任务执行            │  - 自然语言交互      │
│  - 事件管理          │  - 多模态日记          │  - 智能推荐          │
│  - 原材料准备        │  - 成果输出            │  - 跨组件控制        │
│                     │                       │                     │
└─────────────────────┴───────────────────────┴─────────────────────┘
```

### 1.2 组件职责矩阵

| 组件 | 核心职责 | 输入 | 输出 |
|------|----------|------|------|
| Panel | 数据组织与可视化 | 原始记忆数据 | 结构化事件/元素 |
| Canvas | 任务执行与成果生成 | Panel元素 | 多模态成果物 |
| Chatbot | 智能交互与控制 | 用户指令 | 操作指令/内容生成 |

## 3. 核心交互流程

### 3.1 登录流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        登录流程                                  │
└─────────────────────────────────────────────────────────────────┘

用户打开应用
      │
      ▼
┌─────────────┐    无账户    ┌─────────────┐
│  检查登录态  │────────────▶│  注册页面   │
└─────────────┘              └─────────────┘
      │                            │
      │ 有账户                     │ 完成注册
      ▼                            ▼
┌─────────────┐              ┌─────────────┐
│  登录页面   │◀─────────────│  验证账户   │
└─────────────┘              └─────────────┘
      │
      │ 输入凭证
      ▼
┌─────────────┐    失败    ┌─────────────┐
│  身份验证   │───────────▶│  错误提示   │
└─────────────┘            └─────────────┘
      │ 成功
      ▼
┌─────────────┐    首次    ┌─────────────┐
│  检查冷启动  │───────────▶│  冷启动流程 │
└─────────────┘            └─────────────┘

      ▼
┌─────────────┐

└─────────────┘
```

**交互细节：**
- 登录页面：简洁的账户密码输入框，支持第三方登录
- 错误提示：明确的错误信息，提供找回密码入口
- 加载状态：显示加载动画，预估等待时间

#### 4.1.2 Panel 操作交互

| 操作 | 触发方式 | 结果 |
|------|----------|------|
| 缩放进入 | 双指捏合/滚轮/双击 | 进入下一层级 |
| 缩放退出 | 双指展开/滚轮/返回按钮 | 返回上一层级 |
| 拖拽事件 | 长按+拖动 | 移动到Canvas或Chatbot |
| 编辑事件 | 点击事件卡片 | 进入编辑模式 |
| 添加关键词 | 点击"+"按钮 | 弹出输入框 |
| 删除关键词 | 点击"×"按钮 | 移除关键词 |
| 筛选 | 侧边过滤器 | 实时更新显示 |

interface AppState {
  // 用户状态
  user: {
    id: string;

  };

  // Panel状态
  panel: {
    config: ColdStartConfig;
    layout: LayoutType;
    filters: FilterState;
    selectedEvent: Event | null;
    zoomLevel: 1 | 2 | 3;
    events: Event[];
  };

  // Canvas状态
  canvas: {
    activeTask: TaskType | null;
    template: TemplateType;
    elements: CanvasElement[];
    history: HistoryState[];
  };

  // Chatbot状态
  chatbot: {
    messages: Message[];
    context: ConversationContext;
    pendingAction: Action | null;
  };

  // UI状态
  ui: {
    isLoading: boolean;
    activePanel: 'panel' | 'canvas' | 'chatbot';
    notifications: Notification[];
  };
}

## 附录: 交互规范

### A. 响应时间标准

| 操作类型 | 响应时间 | 反馈方式 |
|----------|----------|----------|
| 点击/按钮 | < 100ms | 即时视觉反馈 |
| 页面切换 | < 300ms | 过渡动画 |
| 数据加载 | < 1s | 加载指示器 |
| 数据加载 | > 1s | 进度条 + 预估时间 |
| 拖拽操作 | < 50ms | 流畅跟随 |

### B. 手势支持

| 手势 | Panel | Canvas | Chatbot |
|------|-------|--------|---------|
| 点击 | 选择事件 | 选择元素 | 发送消息 |
| 双击 | 进入编辑 | 放大 | - |
| 长按 | 开始拖拽 | 开始拖拽 | 复制消息 |
| 双指缩放 | 层级切换 | 缩放画布 | - |
| 滑动 | 滚动列表 | 移动画布 | 滚动历史 |

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 新建任务 |
| `Ctrl/Cmd + S` | 保存 |
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Shift + Z` | 重做 |
| `Ctrl/Cmd + F` | 搜索 |
| `Esc` | 取消/返回 |
| `Delete` | 删除选中项 |
